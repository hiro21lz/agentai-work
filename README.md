７歳の子供が考えたシンプルで動くゲームをつくりたい
以下手順を実現するゲームを実装してください。
実装にあたり、仕様を決める必要がある点は質問してください。
参考画像として表示した画像は、ゲームのデザインにそのまま利用してください（特に⑤⑥）

まずカラフルな色の縞々が表示されてる（①）
ユーザーは2色その中から色を選ぶ（Goと書かれているのが、選んだ色）
選んだ2色からランダムな縞ができる（②）
ユーザは色が交互になるように入れ替える
画面の端に◎❌が表示されている（④）
縞々にできたときは、◎を押す
縞々にできないときは、❌を押す
縞々にできるのに❌を押したら、残念な人が表示される（⑤）
同じように、縞々にできないのに◎を推したら、残念な人が表示される
それ以外は、チョコを食べているうれしいひと人（⑥）クリアが表示される
これが一つのステージ、ステージが終われば、次のステージに進む
